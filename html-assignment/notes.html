Notes

title
- Defines a title for the document. It appears in the browser's title bar or in the page's tab.
h1 to h6
- Defines HTML headings from the most important to least important.
strong
- Defines important text. It will appear bold.
img
- Defines an image. The image are linked to web pages.
footer
- Defines a footer for a document or section. The footer element typically contains information, sitemap, back to top links and related documents.
